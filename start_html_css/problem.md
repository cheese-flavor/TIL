### 문제풀면서 개념잡기 ###

#### 기본개념 ####
```
.body, .outer 등은 선택자를 지정하는 개념
그냥 점 없이 진행하면 태그 개념
```

#### 반복개념 ####
```
#가운데 맞춤!

display: flex;
flex-direction: column;
align-items: center;
justify-content: center;
```

#### 중요개념 ####
```
상속자
.선택자 p = 자손결합자 -> 모두를 포함한(넓은범위)
.선택자 > p = 자식결합자 -> 직계 자식만 해당
```
