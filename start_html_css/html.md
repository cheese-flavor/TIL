## 웹 구조화 ##
```
html: 웹페이지의 의미와 구조를 정의하는 언어
markup: 태그 등을 이용하여 문서나 데이터 구조 명시
```

### HTML 구조 ####
``` 
<!DOCTYPE  html> - html 문서라는것을 나타냄
<html> </html> - 전체 페이지의 콘텐츠를 포함
<head> </head> - html 문서에 관련된 설명, 설정 등
<body> </body> - 페이지에 표시되는 모든 컨텐츠

## HTML Element(요소) ##
하나의 요소를 여는 태그와 닫는 태그 
<p> (content) </p>

## HTML Attributes(속성) ##
<p class="editor-note"> -> 이런식으로
규칙 - 속성은 요소 이름과 속성 사이에 공백
    - 하나 이상의 속성들이 있는 경우엔 속성 사이에 공백 구분
    - 속성값은 열고 닫는 따옴표로 감싸기
목적 - 나타내고 싶진 않지만(?) 추가적인 기능, 내용을 담을때
     - CSS에서 해당 요소 선택하기 위한 값 
```


## 웹 스타일링 ##
### html text structure ###
```
HTML 주요 목적 중 하나는 텍스트 구조와 의미 제공
<h1> Heading </h1> : 현재 문서의 최상위 제목(가장 크고 의미부여가 추가된)
- Heading & Paragraphs: h1~6, p(얘는 의미가 멀까?)
- Lists: ol, ul, li
- Emphasis & Importance: em, strong

## CSS ##
```
의미: 웹페이지의 디자인과 레이아웃을 구성하는 언어
구문: 
h1(선택자) {
    color: blue; (선언 -> 속성: 값)
    font-size: 30px;
}

적용방법
1. 인라인 스타일: html 요소 안에 style 속성값으로 작성
2. 내부 스타일: head 태그 안에 style 태그에 작성
3. 외부 스타일: html 링크 태그를 사용해 불러오기
```

### CSS Selectors ###
```
기본선택자
- 전체("*") 선택자: html 모든 요소 선택
- 요소(tag) 선택자: 지정한 모든 태그 선택
- 클래스(class) 선택자: '.' 주어진 클래스 속성 모든 요소
- 아이디(id) 선택자: '#' 주어진 아이디 속성 요소 (하나만 가능)
- 속성(attr) 선택자: 얘는 어디감?

결합자
- 자손 결합자 (" " (space)): 첫번째 요소의 자손 요소들 모두 선택
- 자식 결합자(">"): 첫번째 요소의 직계 자식만 선택
