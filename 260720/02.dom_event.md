### java Script 기본 문법 ###
```
1. 식별자(변수명) 작성 규칙
    - 반드시 문자, 달러($), 또는 밑줄('_')로 시작
    - 대소문자를 구분
    - 예약어 사용 불가 (for, if 등등)

2. 네이밍 케이스
    - 카멜 케이스 = 두단어 작성시 중간 단어 시작 대문자 (변수, 객체, 함수)
    - 케밥 케이스 = '-'로 단어 연결하는
    - 파스칼 케이스 = 앞글자 대문자 (클래스, 생성자)
    - 대문자 스네이크 케이스 = 다 대문자 + 언더바 활용 (상수에 사용)

3. 변수선언 키워드 2가지
    1. let
        - 블록 스코프를 갖는 지역변수 선언
        - 재할당 가능 = 값을 다르게
        - 재선언은 불가능 = 선언을 다르게

    2. const (이거 자주 쓴댕)
        - 블록스코프를 갖는 지역변수를 선언
        - 재할당 불가능 -> 고정값을 갖게 되는!!!!
        - 재선언 불가능 

    * var = 옛날 고대문자/ 존재정도만
    * 선언 시 반드시 초기값 설정 필요!!

4. 블록 스코프
    - if for 함수 등의 중괄호 내부를 가리킴
    - 블록스코프를 가지는 변수는 블록 바깥에서 접근 불가능
```
