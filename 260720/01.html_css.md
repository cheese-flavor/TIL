### HTML 구조 ####
``` 
<!DOCTYPE  html> - html 문서라는것을 나타냄
<html> </html> - 전체 페이지의 콘텐츠를 포함
<head> </head> - html 문서에 관련된 설명, 설정 등 # 링크보낼때 카드 형태로 나오는거!
<body> </body> - 페이지에 표시되는 모든 컨텐츠
<p> </p> - 페이지 본문

## HTML Element(요소) ##
하나의 요소를 여는 태그와 닫는 태그 
<p> (content) </p>
-> 닫는 태그가 없는 경우도 있 = 아예 본문이 필요없거나 등

## HTML Attributes(속성) ##
<p class="editor-note"> -> 이런식으로
규칙 - 속성은 요소 이름과 속성 사이에 공백
    - 하나 이상의 속성들이 있는 경우엔 속성 사이에 공백 구분
    - 속성값은 열고 닫는 따옴표로 감싸기
목적 - 나타내고 싶진 않지만(?) 추가적인 기능, 내용을 담을때
     - CSS에서 해당 요소 선택하기 위한 값

* 이미지 - 서버 기반으로도 가능하고, 상대경로 기반으로도 가능
```

```
## CSS ##
```
의미: 웹페이지의 디자인과 레이아웃을 구성하는 언어
구문: 
h1(선택자) {
    color: blue; (선언 -> 속성: 값)
    font-size: 30px;
}

적용방법
1. 인라인 스타일: html 요소 안에 style 속성값으로 작성 -> 향후 디버깅으로 적용할때
2. 내부 스타일: head 태그 안에 style 태그에 작성
3. 외부 스타일: html 링크 태그를 사용해 불러오기 -> 평소에 진행 / 권장!!
```

### CSS Selectors ###
```
기본선택자
- 전체("*") 선택자: html 모든 요소 선택
- 요소(tag) 선택자: 지정한 모든 태그 선택
- 클래스(class) 선택자: '.' 주어진 클래스 속성 모든 요소
- 아이디(id) 선택자: '#' 주어진 아이디 속성 요소 (하나만 가능)
- 속성(attr) 선택자: 얘는 어디감?

결합자
- 자손 결합자 (" " (space)): 첫번째 요소의 자손 요소들 모두 선택
- 자식 결합자(">"): 첫번째 요소의 직계 자식만 선택

### 명시도 ###
```
명시도: 선택자에서 가중치 계산해서 스타일 적용  -> 겹치는걸 고려하기 위해서
CSS:  웹페이지의 디자인과 레이아웃을 구성하는 언어
Cascade: 동일 가중치일 경우, 마지막에 나오는 선언 사용

##명시도가 높은순서##
1. importance: -!important -> 무조건 우선적용(강제성 때문에 권장 x) / 디버깅
2. inline 스타일
3. 선택자: id 선택자 > class 선택자 > 요소선택자
4. 소스코드 선언 순서

** 요소(태그) 이름은 소문자 사용 / 속성 따옴표는 큰 따옴표 / 에러반환 x
```

## CSS box Model ##
```
모든 html 요소를 사각형 박스로 표현하는 개념
- 내용, 안쪽 여백, 테두리, 외부 간격으로 구성

margin = 가장 바깥쪽, 다른 요소 사이의 공백
border = 콘텐츠와 패딩을 감싸는 테두리
pedding = 콘텐츠 주위에 위치하는 공백
content = 콘텐츠가 표시되는 영역 -> 본문내용

상하좌우 형태로 넓이형태가 먹힘

### 박스 구성의 방향별 명칭 ###
- top: 위 왼 -> 위 오
- right: 위 오 -> 아래 오
- bottom: 아래 오 -> 아래 왼
- left: 아래 왼 -> 위 왼


### width & height 속성 ###
- 요소의 너비와 높이를 지정: 콘텐츠 영역을 대상
- css는 width값을 콘텐츠 크기를 해당 값으로 지정
-> 기준으로 잡는 것에 따라 박스 사이징이 달라짐


### 박스 타입 ###
normal flow = css 적용없이 기본적으로 배치되는 방향


block 타입 특징
- 항상 새로운 행으로 나눔
- width와 height 속성을 사용하여 너비와 높이를 지정

inline 타입 특징
- 새로운 행으로 나뉘지 않음
- width와 height 속성을 사용 x
- 다른 요소를 밀어내는 형태가 아님 -> 수평으로는 밀어낼 수 있음


속성에 따른 수평 정렬
margin-right: auto = 마진 값을 자동으로 한쪽에 치우쳐서
text-aling: left = 텍스트 값을 어느쪽으로 붙일지
margin-right: auto / margin-left: auto =  센터에 공간 남기기

*inline-block: 공간을 만드는것


flex: 메인축을 기반으로 정렬 / 좌에서 우로 / 레이아웃을 잡는
*f12 -> element -> styles -> container
*flexbox forggy -> 개구리 레이아웃 게임

none = 숨김형태 
- 버튼을 눌러서 생기게 할때
- 요소(객체) 존재 -> 공간은 없음? / 따로 데이터를 안받아오는

*visibility = 공간을 차지하게 됨 / 걍 안보이는데 빈공간이 함께 나타남
*opacity = 희미해지게 만드는거
*shorthand = 한번에 지정할 수 있는

4개 = 상우하좌 / 3개 상 좌우 하 / 2개 상하 좌우
*마진상쇄 = 위아래 상쇄는 그냥 합쳐지는
```

## CSS Position ##
```
- 요소를 normal flow에서 제거하여 다른 위치로 배치하는것
- 요소위에 배치를 하려면 겹쳐야하니까 레이아웃이 아닌 형태


유형
페이지 특성 항목의 위치를 조정
1. static: 기본
2. relative: 상대 -> normal flow 기준으로 상대성 / 내 위치를 기준으로 움직이는
3. absolute: 절대 -> 가장 가까운 relative 부모 요소를 기준으로
4. fixed: 고정
5. stickty:
```
