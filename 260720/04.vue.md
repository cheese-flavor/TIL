### VUE ###
```
핵심 2가지 기능
1. 선언적 렌더링: 표준 html을 확장하는 템플릿 구문을 사용하여 선언적으로 작성
2. 반응성: 변경을 추적하고 dom 업뎃

객체에 값을 접근하는 형태는 가능하다

여기에 내용 추가!


기본구조: CreateApp()에 전달되는 객체는 vue 컴포넌트
         setup()함수 내에서 선언되어야 하며 객체를 반환


템플릿 렌더링
- 반환된 객체의 속성은 템플릿에서 사용 가능
- 콧수염 구문을 사용하여 메시 값을 기반으로 동적 텍스트를 렌더링
- 콘텐츠는 식별자나 경로에만 국한되지 않으며, 유효한 자바스크립트 표현식 사용가능
```

class="{active: isactive}" -> 바인딩 해주면 true false에 따라 값 부여가 진행됨 / key:value
ex: <!-- Binding to Objects -->
    <div :class="{ active: isActive }">Text</div>
    <div class="static" :class="{ active: isActive, 'text-primary': hasInfo }">Text</div> -> 여러개 속성 부여가능
    <div class="static" :class="classObj">Text</div> -> 이렇게 클래스 담아서도 진행 가능!

배열을 직접적으로 넣는것도 가능
ex: <!-- Binding to Arrays -->
    <div :class="[activeClass, infoClass]">Text</div>
    <div :class="[{ active: isActive }, infoClass]">Text</div>

```

```

    <!-- event modifiers -->
    <form @submit.prevent="onSubmit">...</form>

```

### 자주 쓰는거3 ###
```
v-if = 표현식 값을 기반으로 true/false
       필요할때 렌더링 하는 -> 토글 비용 up, 실행조건이 변경되지 않는 경우 권장

    <!-- if else -->
    <p v-if="isSeen">true일때 보여요</p>
    <p v-else>false일때 보여요</p> -> 바뀌면 해당 일이 출력

    <!-- else if -->
    <div v-if="name === 'Alice'">Alice입니다</div>  -> else-if 형태
    <div v-else-if="name === 'Bella'">Bella입니다</div>
    <div v-else-if="name === 'Cathy'">Cathy입니다</div>
    <div v-else>아무도 아닙니다.</div>

    <!-- v-if on <template> -->        
    <template v-if="name === 'Cathy'">  -> vue 보여지는 영역만드는! / 랜더링 요소를 아니게끔
      <div>Cathy입니다</div>
      <div>나이는 30살입니다</div>

  </div>
</template>

### 자주쓰는거 4 ###
```
v-for = 소스 데이터 기반으로 요소 또는 템플릿 블록 여러번 렌더링
        for in 형태만 있음
        v-for = "(요소, 인덱스) 형태 in 배열"
        순환 = value, index, key   -> 우선 거꾸로라고 생각하자 **



  <div>
    <div v-for="(item, index) in myArr">
      {{ index }} / {{ item.name }}
    </div>
    <div v-for="(value, key, index) in myObj">
      {{ index }} / {{ key }} / {{ value }}
    </div>

    <!-- v-for on <template> -->      -> 반복문으로 div다 지정할땐 template 쓰기
    <ul>
      <template v-for="item in myArr">
        <li>{{ item.name }}</li>
        <li>{{ item.age }}</li>
        <hr>
      </template>
    </ul>

    <!-- nested v-for -->             -> 중첩 가능
    <ul v-for="item in myInfo">
      <li v-for="friend in item.friends">
        {{ item.name }} - {{ friend }}
      </li>
    </ul>
  </div>


*반드시 v-for와 key를 함께 사용해야함: unique한 식별자여야함
    키값으로 인덱스를 지정하면 안됨: 가변값이니까
    <!-- Maintaining State with key -->
    <div v-for="item in items" :key="item.id">
      <p>{{ item.id }}</p>
      <p>{{ item.name }}</p>
    </div>
    

*v-if와 반복문을 함께 쓸때 = if가 먼저 실행되어서 if를 안으로 넣으면 될듯!

<!-- [Good] v-for with v-if -->
    <ul>
      <template v-for="todo in todos" :key="todo.id">
        <li v-if="!todo.isComplete">
          {{ todo.name }}
        </li>
      </template>
    </ul>

  ```
